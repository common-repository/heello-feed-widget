This plugin allows you to put your heello.com feed into your WordPress Blog / Site by creating a "heello feed" widget.

== Description ==

Now you can put your heello.com feed into your WordPress blog!

Make sure that:

*   Your theme is widget enabled.
*   Your theme stylizes widgets (or write your own css).
*   Your heello account is not private.

== Installation ==

1. Upload `heello.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Draw the "heello feed" widget to an active widget section under "Appearance -> Widgets"
